# 青龙面板 + Art Design Pro 新前端 部署指南

## 一、本地安装（直接运行）

### 1. 构建镜像
```bash
cd qinglong-v2
docker build -t qinglong-v2 .
```

### 2. 运行
```bash
docker run -d \
  --name qinglong \
  -p 5700:5700 \
  -v /opt/qinglong/data:/ql/data \
  qinglong-v2
```

### 3. 访问
浏览器打开 `http://你的服务器IP:5700`


## 二、推送到仓库

### 方案A：Docker Hub（国际通用）

```bash
# 1. 登录（先到 https://hub.docker.com 注册账号）
docker login

# 2. 打标签（把 yourname 换成你的 Docker Hub 用户名）
docker tag qinglong-v2 yourname/qinglong-v2:latest

# 3. 推送
docker push yourname/qinglong-v2:latest
```

### 方案B：阿里云 ACR（国内推荐，速度快）

```bash
# 1. 登录阿里云容器镜像服务
# 访问 https://cr.console.aliyun.com 创建命名空间和仓库
docker login --username=你的阿里云账号 registry.cn-hangzhou.aliyuncs.com

# 2. 打标签
docker tag qinglong-v2 registry.cn-hangzhou.aliyuncs.com/你的命名空间/qinglong-v2:latest

# 3. 推送
docker push registry.cn-hangzhou.aliyuncs.com/你的命名空间/qinglong-v2:latest
```


## 三、服务器安装（从仓库拉取）

### Docker Hub
```bash
docker run -d \
  --name qinglong \
  -p 5700:5700 \
  -v /opt/qinglong/data:/ql/data \
  yourname/qinglong-v2:latest
```

### 阿里云
```bash
docker run -d \
  --name qinglong \
  -p 5700:5700 \
  -v /opt/qinglong/data:/ql/data \
  registry.cn-hangzhou.aliyuncs.com/你的命名空间/qinglong-v2:latest
```


## 四、升级更新

```bash
# 拉取新版本
docker pull yourname/qinglong-v2:latest

# 停止旧容器
docker stop qinglong && docker rm qinglong

# 启动新容器
docker run -d \
  --name qinglong \
  -p 5700:5700 \
  -v /opt/qinglong/data:/ql/data \
  yourname/qinglong-v2:latest
```
> 数据卷 `/opt/qinglong/data:/ql/data` 保持不变，升级不丢数据。


## 五、版本号管理

```bash
# 每次发布打两个标签：版本号 + latest
docker tag qinglong-v2 yourname/qinglong-v2:v1.0.0
docker tag qinglong-v2 yourname/qinglong-v2:latest
docker push yourname/qinglong-v2:v1.0.0
docker push yourname/qinglong-v2:latest
```
